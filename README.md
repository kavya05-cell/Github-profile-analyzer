
# GitHub Profile Analyzer

A simple web application that analyzes GitHub profiles and visualizes developer activity using the GitHub API.

The tool allows users to search any GitHub username and view profile information, repository insights, language usage, and contribution activity in a clean dashboard interface.

---

## Features

• Search any GitHub username
• Display profile information (avatar, name, bio)
• View followers, following, and public repository count
• Show top repositories sorted by stars
• Visualize language usage with a chart
• Display GitHub contribution heatmap
• Responsive dark dashboard UI

---

## Technologies Used

* HTML
* CSS
* JavaScript
* GitHub REST API
* Chart.js
* GitHub Calendar Library

---

## How It Works

1. The user enters a GitHub username.
2. The application fetches profile and repository data from the GitHub API.
3. JavaScript processes the data and dynamically updates the UI.
4. Charts and visualizations present language distribution and contribution activity.
